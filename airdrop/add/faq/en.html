How to Create a Channel and Add vavilon_org_bot

1. Open Telegram and tap the menu icon (three horizontal lines).

2. Select "New Channel".

3. Enter a channel name (e.g., "Airdrop Alerts") and add a description (optional).

4. Set the channel as public or private (your choice).

5. Tap "Create".

6. After creating the channel, go to its settings (gear icon).

7. Select "Administrators" and tap "Add Admin".

8. Search for @vavilon_org_bot and select it from the list.

9. Grant the bot permission to post messages (check "Post Messages").

10. Save the changes. Now the bot can post airdrop alerts in your channel.
